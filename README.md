# StockFlow

Sistema web de gestión de inventario con panel de control, registro de productos, control de stock y alertas de existencias bajas.

## 🔗 Demo

[Ver proyecto en vivo](https://carolrb52.github.io/StockFlow/code.html)

## 🛠️ Tecnologías utilizadas

- **HTML5** — Estructura y contenido de la aplicación
- **CSS3** — Estilos personalizados, animaciones y diseño responsive
- **JavaScript (Vanilla)** — Lógica de la aplicación, manejo de datos y eventos
- **Tailwind CSS** — Framework de utilidades CSS para el diseño visual
- **Google Fonts** — Tipografías Plus Jakarta Sans e Inter
- **Material Symbols** — Iconografía de la interfaz

## 📋 Características

- Panel de control con métricas en tiempo real
- Registro y gestión de productos
- Control de stock con alertas de existencias bajas
- Búsqueda y filtrado de inventario
- Modo oscuro / claro
- Diseño responsive para todos los dispositivos
- Modales de creación y edición de productos
- Notificaciones toast
